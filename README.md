# API для подачи и обработки заявки на займ 

## Запуск проекта
``cp .env.example .env``  
``docker compose up -d --build ``   
``docker compose exec app sh -c "php yii migrate" ``

Затраченное время 3-4 часа

По тз реалиовал как синхронно с эмулирированием продолжительного времени принятия решения каждой заявки. Но думаю что такая реализация в корне не верная и сам я бы реализовал с помощью очередей и фоновых процессов. Остаавил такое решение из-за тестов которыми будут проерят тестовое задание
